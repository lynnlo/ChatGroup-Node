Chat Group Node

A multitool bot for discord.

Created by Mario O, Lynn O, Andres A.


Project Details

ChatGroup Node is an open sourced project aimed at making a extensive and usefull collecction of commands and tools.

Want to join? Check out our github!


Features

ChatGroup Node comes with many out of the box tools.

  1. Calculator

    An easy to use multifunction calculator with advaced equations.

  2. Economy

    A coin based economy with your coins saving across all servers.

  3. Javascript Evaluator

    A full javascript evaluator, great for running simple programs.

  4. API Connectivity

    Commands for getting images and data from serveral REST APIs.
